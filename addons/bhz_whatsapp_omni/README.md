# BHZ WhatsApp Omni

- O módulo lê automaticamente as variáveis `STARTER_BASE_URL` (ou sinônimos `RENDER_STARTER_URL`, `STARTER_SERVICE_URL`) e `ODOO_WEBHOOK_SECRET`/`STARTER_WEBHOOK_SECRET` durante a instalação.
- Com isso, o cliente não precisa configurar parâmetros manualmente no Odoo.
- Após instalar/atualizar, basta clicar em **Conectar via QR** na conta Starter para parear o WhatsApp.
