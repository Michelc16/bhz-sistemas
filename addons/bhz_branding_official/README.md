# BHZ Branding Oficial

Módulo para personalizar o Odoo 19.0 (Odoo.sh / Enterprise) com a identidade BHZ SISTEMAS.

## O que faz

- Troca o título da aba do navegador para "BHZ SISTEMAS".
- Troca o favicon.
- Remove/neutraliza as views de promoção de marca da Odoo (`web.brand_promotion*`, `website.brand_promotion`, `website_sale.brand_promotion`).
- Troca o logo do cabeçalho do website pela logo BHZ.

## Como usar

1. Coloque a pasta `bhz_branding_official` no seu diretório de addons.
2. Faça commit/push para o repositório do Odoo.sh.
3. Atualize a lista de apps e instale **BHZ Branding Oficial**.
4. Limpe o cache do navegador.
